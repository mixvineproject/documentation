Документация проекта

Изменения версий

История изменений проекта:

web0dev (2024)

Начальная версия. Создание базовой структуры сайта на GitHub Pages. Основное внимание уделялось написанию HTML-кода. Простой, но функциональный интерфейс для тестирования базовых элементов взаимодействия.

web0.1dev (2024)

Первая версия интерфейса с добавлением кнопок для пользовательского взаимодействия. Улучшение удобства навигации по сайту.

web0.5Happy (2024)

Эксперименты с кодом, добавление развлекательных элементов для повышения привлекательности и вовлеченности пользователей.

web1dev (2025)

Начало разработки кейса с анимацией и системой получения призов. Добавление интерактивности и элемента неожиданности.

web1.1dev (10.04.2025)

Глобальный патч. Устранение множества ошибок и улучшение функциональности. Внедрение рабочей рулетки для азартных игр и получения призов. Стабильность и удобство использования.

web2Global (11.04.2025)

Полная переработка интерфейса для более современного дизайна. Рабочий кейс стал более интерактивным с анимациями. Специальные эффекты для редких предметов.

web3Global (11.04.2025)

Введение главного меню для быстрого доступа к кейсу, Telegram-каналу, боту и разработчику. Добавлены Политика конфиденциальности и Пользовательское соглашение. Cooldown (таймер) на 30 минут перед открытием кейса.

web3.1Global (12.04.2025)

Реализация глобального патча с регулярными обновлениями, новый проект "Рандомайзер". Обновление главного меню. Два новых особых предмета в кейсе. Убран Cooldown для неограниченного открытия кейса.

web4Global (12.04.2025)

Глобальное обновление проекта, развивающегося усилиями одиночки, не обладающего глубокими знаниями программирования.

  • Обновления Рандомайзера: Расширение функциональности Рандомайзера: "Исключение повторов", функция "По порядку", режим "Розыгрыш".
  • Упрощение системы кейса: Упрощение архитектуры кейса. Открытие кейса без ограничений (требуется перезагрузка страницы из-за багов).
  • Обновление главного меню: Новые подразделы: "Крупные проекты", "Некрупные проекты", "Информация и поддержка".
  • Введение страницы с документацией: Страница с документацией для отслеживания изменений и версий проекта.
